Unbabel Front-end Starter

Welcome! Bootstrap is ready to use.

This project attempts to be a quick and easy to setup environment for front-end development at Unbabel.

It ships with Bootstrap (v4.0.0-alpha.6) to help you get started quickly. View Documentation

It includes a CSS structure that is easy to understand if you need to develop custom stuff.

Bootstrap is easy to remove, just run yarn remove bootstrap, then go to src/scss/_dependencies.scss and remove bootstrap.

Contribute to this project at unbabel/frontend-starter
